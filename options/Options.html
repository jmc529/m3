Key Bindings

Key bindings are preset to:
Shuffle: "Ctrl+Shift+End"
Previous: "Ctrl+Shift+Insert"
Play/Pause: "MediaPlayPause"
Next: "Ctrl+Shift+Delete"
Repeat: "Ctrl+Shift+Home"

To set a new key shortcut select a combination suitable for your needs and save. Unfortunately only one mediaKey can be assigned due to a bug in Firefox.

Premium Caveats

Spotify has a wall on their API that forbids non-premium users from using all of it's features. This includes commonplace features like repeat and shuffle as well as nuanced features like connecting to a device. This app employs a simple workaround but it requires the open.spotify.com page to be up.

If you would like to use these extra features with out premium it is suggested you keep spotify open in a pinned tab. If you are interested in spotify premium click this link.

You can set spotify to open up in a pinned tab on startup. It is off by default.
